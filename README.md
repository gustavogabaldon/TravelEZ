# TravelEZ

TravelEZ is a web-based application designed to assist students in managing their budgets for study abroad programs. It features tools for estimating and tracking expenses, including accommodation, food, travel, and other costs.

## Features

- Input and categorize expenses (tuition, housing, food, travel, activities, etc.)
- Provide expense estimates based on the destination country and duration of stay
- Track actual expenses and compare them against the budget
- Offer insights and tips on saving money while studying abroad

**Prerequisites**
Make sure you have the following software installed on your machine:

Python 3.x
Flask
Pandas

## Installation

To set up TravelEZ on your local machine, follow these steps:

1. Clone the repository:
git clone https://github.com/your-username/TravelEZ.git
cd TravelEZ
pip install -r requirements.txt

**Usage**
Run the Flask web application:
python app.py
